A stunning, responsive, and highly interactive agency portfolio website built using **Next.js 15 (App Router)**, **TypeScript**, and **Tailwind CSS**. Designed to showcase projects, services, and team details with fluid, high-performance animations.

---

## ✨ Features

- **🎯 Modern UI/UX:** A clean, minimal, and professional design crafted to engage visitors.
- **📱 Ultra-Responsive:** Optimized for flawless viewing across mobile, tablet, and desktop screens.
- **🎬 Fluid Animations:** Powered by **Framer Motion** for smooth, delightful transitions and hover effects.
- **🎨 Custom shadcn/ui:** Tailored, accessible UI component patterns built on top of Tailwind CSS.
- **🛡️ Fully Type-Safe:** Built with strict TypeScript for robust, error-free development.
- **⚡ Next.js 15 Power:** Leverages the latest React 19 features, lightning-fast rendering, and optimized performance.

---

## 🛠️ Tech Stack

| Technology | Purpose |
| :--- | :--- |
| **Next.js 15** | Framework (App Router, Server & Client Components) |
| **TypeScript** | Type-safe JavaScript superset |
| **Tailwind CSS** | Utility-first rapid styling |
| **Framer Motion** | Physics-based micro-interactions and page transitions |
| **Lucide React** | Consistent, lightweight vector iconography |
| **shadcn/ui** | Base component architecture patterns |
